GrokProxy

A serverless proxy that converts Grok API calls to OpenAI API format.

Note: This is an API endpoint, not a website. You won't see much here in your browser.

How to Use

Set your OpenAI API base URL to:

https://your-deployment-url.com/v1

Then use your Grok API key as the OpenAI API key in your application.

Supported Endpoints

  • /v1/chat/completions - Chat completions API
  • /v1/completions - Text completions API
  • /v1/embeddings - Text embeddings API
  • /v1/models - List available models

Environment Variables

  • GROK_API_KEY - Your Grok API key (if not provided in requests)
  • GROK_API_BASE - Optional override for Grok API base URL
